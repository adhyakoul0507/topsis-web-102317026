## Project Overview

This project implements a web-based version of the **TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)** method.

The user can:

- Upload an input file (.csv)
- Provide weights
- Provide impacts (+ or -)
- Enter email ID
- Receive the TOPSIS result file via email
